## How to Add Categories in Sanity:

1. Open your Sanity Studio (run `cd sanity_ecommerce` then `npm run dev`)
2. Click on "Category" in the sidebar
3. Click "Create new" button
4. Fill in the fields:
   - **Category Name**: Enter the name (e.g., "Smartphones")
   - **Slug**: Click "Generate" to auto-create from name
   - **Description**: Enter a brief description
   - **Category Image**: Upload an image (recommended 250x250px or larger)
5. Click "Publish"

## How to Add Products in Sanity:

1. Open your Sanity Studio (run `cd sanity_ecommerce` then `npm run dev`)
2. Click on "Product" in the sidebar
3. Click "Create new" button
4. Fill in all required fields:
   - **Name**: Product name (e.g., "iPhone 15 Pro Max")
   - **Slug**: Click "Generate" to auto-create from name
   - **Price**: Enter the price as a number (e.g., 1199)
   - **Details**: Enter product description
   - **Category**: Select a category (REQUIRED)
   - **Product Type**: Select a product type (OPTIONAL - only shows types for selected category)
   - **Image**: Upload product image(s) - you can add multiple images
5. Click "Publish"

**Note**: Make sure categories and product types are created first before adding products!
